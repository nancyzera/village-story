How to Add Your OpenAI API Key Securely

For security reasons, API keys are stored in Replit Secrets, not in the application code. Follow these steps:

  1. 1 Click the Secrets tab in the left sidebar of Replit
  2. 2 Click "+ New Secret"
  3. 3 Set the Key to: OPENAI_API_KEY
  4. 4 Paste your OpenAI API key in the Value field
  5. 5 Click "Save" and the app will automatically restart
